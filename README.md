# おやしろさまショック — Oyashiro-sama Shock Translations

A reference tool for browsing translated quiz questions and answers from **Shin Quiz Shock: Oyashiro-sama Shock** (新・クイズ・ショック おやしろさまショック).

Each entry includes the original Japanese text, English translation, romaji readings, the correct answer, wrong answers, and a screenshot from the game.

## Features

- Browse all quiz entries organized by level
- View all levels at once with the "A" (all) tab
- Star/unstar entries to filter by importance
- Trilingual display: Japanese, English, and romaji
- Offline support (PWA) — works without internet after first visit
